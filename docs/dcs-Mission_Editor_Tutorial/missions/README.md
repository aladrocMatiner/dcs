# Missions folder (optional)

This folder is for:

- A “course base mission” per chapter (so you can compare progress).
- A set of tiny recipe missions (`recipes/*.miz`) to copy/paste from.

Suggested naming:

- `course_base_ch01.miz`, `course_base_ch02.miz`, …
- `recipes/zone_message.miz`
- `recipes/f10_start_menu.miz`
- `recipes/3_phase_flags.miz`
