## Common errors and fixes

- **I get a `format` error**:
  - Check parentheses.
  - Make sure `%d` / `%.1f` matches what you pass in.
- **I canâ€™t print a `%` sign**:
  - Use `%%` inside `string.format`.

> **If you get an error, do this:**
> - Start very small:
>   - `local t = string.format("x=%d", 1)`
>   - `trigger.action.outText(t, 10)`
