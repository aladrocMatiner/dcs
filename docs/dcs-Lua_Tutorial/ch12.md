## Common errors and fixes

- **Counter never increases**:
  - The event `S_EVENT_KILL` might not be happening.
  - Try a mission with an easy static target and destroy it.
- **Too many messages**:
  - Reduce message duration or throttle events later.

> **If you get an error, do this:**
> - Keep the handler, but replace logic with one fixed message to prove it triggers.
