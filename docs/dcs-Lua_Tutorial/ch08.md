### Step 1: set the flag in the editor (no code)

## Common errors and fixes

- **It always says OFF**:
  - Make sure the flag trigger runs earlier (time order matters).
  - In DCS, flags are commonly numeric `0/1` (not true/false).
- **Flag name confusion**:
  - `trigger.misc.getUserFlag("1")` uses the flag â€œnameâ€. Here it is `"1"`.

> **If you get an error, do this:**
> - Use a `DO SCRIPT` probe:
>   - `trigger.action.outText("Flag1=" .. trigger.misc.getUserFlag("1"), 10)`
