## Quick test (step-by-step)

Goal: see a big message on screen that proves Lua executed.

### Option A (simplest): `DO SCRIPT` (paste)

1) Open DCS â†’ *Mission Editor*.
2) Create a new mission (one player aircraft on the runway is enough).
3) Go to *Triggers*.
4) Create a trigger:
   - **Type**: `MISSION START`
   - **Action**: `DO SCRIPT`
5) Paste:

```lua

```

6) Save the mission.
7) Run the mission.

### Option B (better long-term): `DO SCRIPT FILE` (file)

1) In Mission Editor â†’ *Triggers* â†’ `MISSION START`.

3) Select: [examples/ex03_hello.lua](examples/ex03_hello.lua)
4) Save the mission (DCS usually packages the script into the `.miz`).
5) Run the mission.

## Common errors and fixes

- **I donâ€™t see the message**:
  - Make sure the trigger type is `MISSION START`.
  - Make sure the duration is `10` (seconds).
- **Error / nothing happens**:
  - Start with `DO SCRIPT` (Option A). It removes file handling from the equation.
  - If you use `DO SCRIPT FILE`, re-select the file and save again.

> **If you get an error, do this:**
> - Reduce to one line (`outText` only).
> - Save, run, verify, then add more.
